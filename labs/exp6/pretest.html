Application of basic logic gates in fire and burglar alarms

Pre Test

Q1. Which gate is known as ‘any or all gate'?
A.     OR gate
B.     AND gate
C.     EX-OR gate
D.     NOT gate
Ans is A

Q2. Select the incorrect input and output pin combination in IC 7432
A.     Input 1, 2 output 3
B.     Input 4, 5 output 6
C.     Input 8, 9 output 10
D.     Input 13, 12 output 11
Ans is C

Q3. Which gate is known as ‘all or nothing gate’ ?
A.     OR gate
B.     AND gate
C.     XNOR gate
D.     NAND gate
Ans is B

Q4. Indicate which of following gates can be used to realize all the possible combinational logic functions
A.     OR gate only
B.     AND gate only
C.     NOR gate only
D.     Ex-OR gate only
Ans is C

Q5. How do you make an AND gate out of a NAND gate using inverters (NOT gates)?
A.     Invert the output from the NAND gate
B.     Invert both the inputs to the NAND gate
C.     Invert one of the inputs to the NAND gate
D.     Invert both the inputs and output of the NAND gate
Ans is D

Q6. Which of the following gates is known as coincidence detector?
A.     AND gate
B.     OR gate
C.     NOT gate
D.     NAND gate
Ans is A

Q7. For a Boolean expression Y=A’(AB’+A’), the output is high
A.     When input B is high
B.     When input B is low
C.     It is independent of input B
D.     None of the above
Ans is C

Q8. The Boolean expression for the output F1 is F1=XYZ' + XY'Z + X'YZ + X'Y'Z'. This is ________ form.
A.     Product of sum
B.     Sum of product
C.     Straight
D.     None of the above
Ans is B

Q9. The only function of NOT gate is to __________
A.     Stop signal
B.     Invert input signal
C.     Act as a universal gate
D.     None of the above
Ans is B

Q10. What type of logic gate's behavior does this truth table represent?

A.     2 input OR gate
B.     3 input OR gate
C.     3 input EXOR gate
D.     4 input EXOR gate
Ans is B
