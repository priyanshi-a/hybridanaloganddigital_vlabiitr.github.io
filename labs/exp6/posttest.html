Application of basic logic gates in fire and burglar alarms

Post Test

Q1. A positive AND gate is also a negative
A.  NAND gate
B.  NOR gate
C.  AND gate
D.  OR gate
Ans is D

Q2. The Boolean expression for two input NAND gate
A.  (A.B)’
B.  (A+B)’
C.  A’.B’
D.  A+B
Ans is A

Q3. For multi-input AND gate, the unused input pins should not be left_____.
A.  On
B.  Connected
C.  Unconnected
D.  None of the above
Ans is C

Q4. A NOT gate IC 7404 contains ______ NOT gates.
A.  3
B.  4
C.  6
D.  2
Ans is C

Q5. What is the logic status at A and B to activate the alarm in fire alarm system where A represents temperature sensor while B represents smoke sensor
A.  A=1,B=0
B.  A=0,B=1
C.  A=1,B=1
D.  All of the above
Ans is D

Q6. A fire alarm system can be implemented using
A.  Basic logic gates
B.  Op-Amp comparator
C.  555 timer IC
D.  All of the above
Ans is D

Q7. The Boolean expression in term of A, B and C to activate the buzzer in burglar alarm system where A represents arm key switch, B represents Reed switch and C represents pressure mat, is
A.  A’.B.C
B.  A.B.C
C.  A.(B+C)
D.  A.(B’+C)
Ans is D

Q8. How do you make a NAND gate out of an AND gate using inverters (NOT gates)?
A.  Invert the output from the AND gate
B.  Invert both the inputs to the AND gate
C.  Invert one of the inputs to the AND gate
D.  Invert both the inputs and output of the AND gate
Ans is A

Q9. The output of the logic gate in figure is

A.  0
B.  1
C.  A'
D.  A
Ans is C

Q10. A certain inverting amplifier has a closed loop voltage gain of 25. For the logic circuit shown in the figure, the required input condition (A,B,C) to make the output X =1 is

A.  1, 0, 1
B.  0, 0, 1
C.  1, 1, 1
D.  0, 1, 1
Ans is C
