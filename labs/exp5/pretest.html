Bistable Multivibtator using 555 Timer IC

Pre Test

Q1. Circuit which produces an output wave shape resembling that of a symmetrical or asymmetrical square wave is called
A.     Timing generator
B.     Frequency generator
C.     Multivibrator
D.     Single vibrator
Ans is C

Q2. Multivibrator in which circuit is stable in either state and can be flipped from one state to other by an external trigger pulse is termed as
A.     Astable multivibrator
B.     Monostable multivibartor
C.     Bistable multivibrator
D.     Tristable multivibrator
Ans is C

Q3. Multivibrator having a quasi-stable state is called
A.     Bistable multivibrator
B.     Monostable multivibrator
C.     Astable multivibrator
D.     None of the above
Ans is B

Q4. The multivibrator which generates square wave of its own is the _________ multivibrator.
A.     Monostable
B.     Bistable
C.     Astable
D.     None of the above
Ans is C

Q5. A multivibrator is an electronic circuit used to implement ____________.
A.     Oscillator
B.     Timer
C.     Flip-Flop
D.     All of the Mentioned
Ans is D

Q6. Bistable multivibrator can be designed by using
A.     Transistors
B.     555 timer IC
C.     Both options A and B
D.     None of the above
Ans is C

Q7. The internal circuitry of the 555 timer consists of ___________, an R-S flip-flop, a transistor switch, an output buffer amplifier, and a voltage divider.
A.     A mutiplexer
B.     A voltage amplifier
C.     Two comparators
D.     A peak detector
Ans is C

Q8. In 555 timer IC, pin no. 3 is _______ terminal of the IC
A.     Trigger
B.     Output
C.     Threshold
D.     Discharge
Ans is B

Q9. In a Bistable multivibrator circuit, a commutating capacitor is used
A.     To increase the base storage charge
B.     To provide ac coupling
C.     To increase the speed of response
D.     To provide the speed of oscillations
Ans is C

Q10. Capacitor discharge interval in monostable circuits is known as
A.     Refresh time
B.     Recovery time
C.     Dynamic time
D.     Static time
Ans is B
